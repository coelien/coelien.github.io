# Notes on Missing semester

## environment variables

- PATH

shell搜索程序时，会去寻找PATH列表的所有路径下匹配的程序

## 实用命令

- which

显示所运行程序的所在路径

- cd -

回到上次所在目录，非常好用！

- ls ..

显示上层的所有文件

- ctrl - l

清除屏幕，光标回到顶部
